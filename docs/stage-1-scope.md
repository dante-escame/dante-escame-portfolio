# Stage 1 Scope

## Goal
Stage 1 establishes the real portfolio project baseline. The focus is on inheriting the approved stack and visual foundation from `style-testing`, then reframing that foundation around a professional personal portfolio build.

## Included In Stage 1
- Next.js 16 App Router bootstrap with React 19 and TypeScript
- Tailwind CSS v4 setup and global design tokens in `app/globals.css`
- Base route layout and initial homepage scaffold
- Reusable UI components and Ladle-based component documentation setup
- AI guidance files and frontend architecture guidelines adapted to this repository
- Docker standalone build path
- Initial project README and stage documentation

## Not Included Yet
- Final portfolio information architecture
- Real content, case studies, project data, and copywriting
- Production-ready accessibility review
- Analytics, forms, CMS, or database-backed features
- Automated tests beyond baseline guidance
- Deployment configuration beyond the Docker and Next.js foundation

## Stage 1 Exit Criteria
- The repository installs and builds with the approved stack.
- Shared design tokens and reusable UI primitives are present in the new codebase.
- The homepage clearly presents the project as a portfolio foundation instead of a style sandbox.
- AI docs and workflow guidelines are aligned with the new repository purpose.
